===================
3. Организуйте хранение категорий и товарных позиций учебной базы данных shop в СУБД MongoDB.
===================

По совету Игоря в видео уроке использую для этой задачи графический клиент robomongo, который теперь называется 3T Robo.
Скачиваю клиент с оф-сайта https://robomongo.org/download.

Разворачиваю в docker сервер mongodb 
>sudo docker run -p 27017:27017 --name some-mongo -d mongo

Создаю connection в Robomongo, используя настройки сервера mongo.

Создаю подключение к MySQL.

Создаю базу данных shop, и по очереди (может можно и все сразу, не разобралась с ходу) импортирую таблицы в базу как отдельные 
коллекции: users,orders,catalogs,products и т.п.

Вижу кнопку SQL, пробую создать запрос, работает так же как SQL.

Сделаю Export базы данных shop как набор JSON документов, типа dump получается.
